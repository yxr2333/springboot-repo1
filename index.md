# 一、项目描述
三峡大学帮扶系统是帮助三峡大学计算机与信息学院学生会的学风建设部管理学业帮扶活动的一个项目。其内容大概分为以下几个部分。
## 1.1 网站展示部分
构想大概为，此模块用于部门展示以及前期的报名工作（小导师的报名和学员的报名都可以在此进行）。后台可以对报名的小导师和学员进行一系列的CURD操作

## 1.2 核心管理部分
对录用的小导师和报名成功的学员进行管理。比如开设班级，给班级分配小导师和学员，设置帮扶科目，让干事对接每一个班级等等操作。
